Trello List Exporter

A privacy-first Trello Power-Up for exporting list cards to JSON and CSV.

MVP 0: planning and project foundation

The Power-Up is not registered yet. This page exists so GitHub Pages has a stable HTTPS root while the implementation is built.

  • Exports will run in the browser for early MVPs.
  • No Trello board data should be committed to this public repository.
  • No backend or third-party data processor is planned for the initial build.

Project details are tracked in the implementation plan.
